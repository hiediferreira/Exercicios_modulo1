** Removendo atributo**

Exercício 09 - Remova um atributo de um elemento na página HTML.

Este é um exercício simples para exercitar a remoção de atributos. Quando o mouse clicar na div, o atributo id="text" será removido, removendo as estilizações do elemento "p".
